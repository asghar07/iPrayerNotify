iPrayerNotify

Titanium application - calculates the prayer time of the current or searched location based on different methods of calculation.

Islamic Prayer Time

This Titanium Mobile application tested on iPhone is available in itunes store, is used to check the islamic prayer timing for a particular location. The source code is made available as open source.
